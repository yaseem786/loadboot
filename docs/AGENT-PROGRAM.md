# LoadBoot Agent Program — "Build a chain. Earn on every load. Forever."
(Independent dispatchers ke liye — ready-to-send email neeche section 7 mein hai)

## 1. The model in one line
Aap LoadBoot par clients ke JORE (pairs) lao — broker+carrier, ya shipper+broker —
aur un ke beech jo bhi load DELIVER ho, uska **1% aap ka**, har load par, hamesha.

## 2. Kyun pair (jora)?
Ek akela broker board par load post karega lekin book kaun karega? Ek akela carrier
board dekhega lekin load kahan se aayega? **Pair = zinda marketplace.** Isi liye:
- Sirf broker refer kiya = abhi kuch nahi (pending)
- Broker + carrier (ya shipper + broker) = CHAIN ACTIVE → ab kamai shuru
- Chain active hone ke baad aap ke KISI BHI referred client ke har delivered load par 1%
  (misal: aap ka broker load post kare aur koi BHI carrier book kare — phir bhi 1%,
  kyunke aap ki chain active hai)

## 3. Flow — bilkul automatic
1. Aap ko LoadBoot AGENT ACCOUNT milta hai (login + apna dashboard)
2. Dashboard mein aap ka REFERRAL LINK — har client isi link se join karwao
3. Join hote hi aap ke dashboard par LIVE dikhta hai: "Broker X joined ✓"
4. Broker load post karta hai → aap dekh sakte ho; carrier book karta hai → GPS tracking
   aap ke dashboard par bhi live
5. Trip DELIVER hota hai → **1% aap ke account mein land** — koi invoice, koi chasing nahi;
   software khud hisaab karta hai (wahi engine jo carrier/broker ka paisa track karta hai)
6. Payout monthly — Payoneer/bank transfer, $100 minimum balance par

## 4. Aap ka kaam (yehi "job" hai)
- Apne laye hue clients ke DEDICATED DISPATCHER/success agent bano
- Dashboard se live chat par unki madad karo (onboarding, pehla load, documents)
- Jitna aap ka pair active rahega, utni aap ki recurring kamai — isliye unhe zinda rakhna
  aap ke faide mein hai. Software sab operations khud karta hai (booking, tracking,
  detention proof, invoicing, payments) — aap sirf rishta sambhalo.

## 5. Ek misal — asli numbers
Aap ne 1 broker + 2 carriers join karwaye (chain active).
Broker mahine ke 20 loads post karta hai, average $2,400/load, sab deliver hue:
- 20 × $2,400 × 1% = **$480/mahina** — sirf EK chain se
- 5 aisi chains = **$2,400/mahina**, recurring, jab tak clients platform par hain
Koi cap nahi. Jitni chains, utni kamai.

## 6. Rules (saaf-saaf)
- 1% GROSS load value par, sirf DELIVERED loads par (GPS-verified delivery — jo trip
  poora nahi hua uska kuch nahi)
- Chain active hone ki shart: kam az kam ek PAIR (broker+carrier ya shipper+broker)
- Referral link se hi join hona zaroori (baad mein claim nahi hota — link hi record hai)
- Client pehle se LoadBoot par ho to uska referral count nahi hota
- Self-referral / fake accounts = program se bahar
- LoadBoot apni 5% dispatch fee alag kamata hai — aap ka 1% us se alag hai, carrier ya
  broker ki jeb par koi asar nahi
